# 🎮 Modern Tic-Tac-Toe

[![Live Demo](https://img.shields.io/badge/Live-Demo-blue)](https://ashwinwagh3.github.io/responsive-tic-tac-toe/)
[![MIT License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

A beautifully crafted, fully responsive Tic-Tac-Toe game with a modern glass-morphism design. Play against a friend on any device while tracking your scores in style.

## ✨ Features

- 🌙 **Dark Mode Toggle** – Seamlessly switch between light and dark themes (preference saved locally)
- 📊 **Score Tracking** – Keep track of wins for X, O, and draws in real-time
- 🎯 **Win Highlighting** – Winning cells are visually highlighted for instant feedback
- 🔄 **Game Controls** – Reset the current game or clear all scores with dedicated buttons
- 📱 **Fully Responsive** – Plays perfectly on desktop, tablet, and mobile devices
- 💎 **Modern UI** – Glass-morphism effects, smooth animations, and clean typography

## 🎲 How to Play

1. The game starts with **X**'s turn
2. Click any empty cell to place your mark
3. The game automatically detects wins or draws
4. Scores update instantly after each round
5. Use **"New Game"** to continue playing or **"Reset Scores"** to start fresh

## 🛠️ Technologies

- **HTML5** – Semantic structure
- **CSS3** – Custom properties, flexbox, grid, backdrop blur, transitions
- **Vanilla JavaScript** – No frameworks, pure DOM manipulation
